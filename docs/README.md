# Life OS Platform

Plateforme de productivité personnelle et de bien-être. Centralisez tâches, agenda, objectifs, humeur et sessions focus en un seul endroit.

## Modules

| Module | Description |
|---|---|
| **Tableau de bord** | Vue d'ensemble des tâches actives, priorités hautes et suggestions IA |
| **Tâches** | Gestion avec vue liste et Kanban, filtres par statut/priorité |
| **Agenda** | Calendrier mensuel avec événements |
| **Objectifs** | Suivi d'objectifs avec progression 0-100% |
| **Bien-être** | Suivi d'humeur avec score 1-10, historique et graphique |
| **Focus** | Sessions de travail chronométrées |
| **IA** | Suggestions personnalisées et rapport hebdomadaire |
